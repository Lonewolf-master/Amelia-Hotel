# Specification: Immersive Visuals & Search Optimization

This track focuses on enhancing the user's visual journey and ensuring the site is technically sound for search engines and high performance.

## 1. Immersive Room Lightbox
- **Goal:** Allow guests to view high-resolution imagery of each room type.
- **Features:**
    - Full-screen modal with GSAP-animated entry/exit.
    - Image slider for room-specific galleries.
    - Pre-selection logic: Clicking "View" opens the lightbox; "Book" within lightbox triggers the booking flow.
- **Aesthetic:** Minimalist, high-contrast dark overlay with gold accents.

## 2. SEO & Technical Performance
- **Goal:** Improve search visibility and site speed.
- **Features:**
    - **Meta Management:** Dynamic titles and descriptions per page/state.
    - **Image Performance:** Lazy loading and modern attribute usage.
    - **Sitemap & Robots:** Technical files for search engine crawlers.
    - **Accessibility Audit:** Ensure ARIA roles and contrast compliance.

## 3. Visual Polish
- **Smooth Scroll:** Fine-tune scroll behavior for a "luxury feel".
- **Asset Cleanup:** Replace any remaining placeholders with high-fidelity authentic images.
