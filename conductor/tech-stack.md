# Tech Stack

## Core Technologies
- **Frontend Framework:** React (TypeScript)
- **State Management:** React Context API
- **Styling:** 
    - Tailwind CSS (for layout and utility-first styling)
    - CSS Modules (for component-specific styling)
    - Vanilla CSS (for fine-tuned luxury animations and overrides)

## Libraries & Tools
- **Animations:** GSAP (GreenSock Animation Platform)
- **Maps:** Google Maps Platform API (for embedded interactive maps)
- **Icons:** Lucide React or similar premium-feel icon library
- **Build Tool:** Vite (for fast development and optimized production builds)

## Development Standards
- **Language:** TypeScript (for type safety and better developer experience)
- **Formatting:** Prettier
- **Linting:** ESLint
